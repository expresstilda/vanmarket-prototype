# Ван Маркет — прототип сайта

Интерактивный прототип нового сайта **ООО «ТД «Ван-Маркет»** — российского производителя нетканых материалов (спанбонд, гидропароизоляция ИЗОБОНД, диффузионные мембраны, геотекстиль, полиэтиленовая плёнка, укрывные материалы).

## Живая версия

👉 **https://expresstilda.github.io/vanmarket-prototype/**

## Что это

Кликабельный многостраничный прототип (одностраничное приложение с маршрутизацией): каталог продукции с карточками товаров, отраслевые решения, блок «замкнутый цикл», «под своим брендом», дилерская программа, документы, блог, где купить, вакансии, контакты.

Прототип открывается в светлой и тёмной теме, адаптивен под мобильные устройства.

## Статус данных

- Тексты, категории, характеристики, представители, вакансии — по данным действующего сайта van-market.com и брифа.
- **Цены** — «по запросу» (на сайте компании цены не публикуются); будут подставлены после утверждения прайса.
- Фото, номера сертификатов и точные ТХ по артикулам — подставляются после передачи материалов клиентом.

Формы в прототипе работают в демо-режиме (заявки не отправляются).
